⚙ Tetapan Google Drive

Masukkan URL deployment Google Apps Script yang dah dikonfigurasi dengan akaun edu.my awak.

Dapatkan URL ini dari: Apps Script → Deploy → Manage Deployments

Cara setup Apps Script:

  1. Pergi ke script.google.com guna akaun edu.my
  2. Buat projek baru → tampal kod Code.gs
  3. Klik Deploy → New deployment → Web app
  4. Execute as: Me | Access: Anyone (atau domain sahaja)
  5. Salin URL dan tampal di atas
